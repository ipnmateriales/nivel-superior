# 📦 Descargar el Repositorio Curso NS

Este repositorio contiene todos los archivos necesarios del proyecto.

Puedes descargar todos los archivos de dos maneras:

## ⬇️ Opción 1: Descargar como ZIP (Más fácil)

1. Haz clic en el botón verde **Code**.
2. Selecciona **Download ZIP**.
3. Extrae el archivo en tu computadora.

## 💻 Opción 2: Clonar el repositorio con Git

Si tienes Git instalado, puedes descargar el proyecto usando:

```bash
